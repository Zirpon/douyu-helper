## 🛠️ 描述

-   进入 `浏览器插件市场` 安装 `油猴/Tampermonkey` 插件
-   点击油猴插件 ~ 获取新脚本 点击 [GreasyFork](https://greasyfork.org/zh-CN)
-   搜索 【`DOTA2 暗黑狂欢商店 Dark Carnival Store`】~ 安装脚本
-   安装成功后 手动打开 完美 `DOTA2` 维基官网页面(https://wiki.dota2.com.cn/)
-   首次打开网页 等待 10 秒 网络加载保存在 github 的 票券图片资源
    ![](https://raw.githubusercontent.com/Zirpon/douyu-helper/main/src/dota2_darkcarnival/assets/img/all_token.png)
-   选择代币
    ![](https://raw.githubusercontent.com/Zirpon/douyu-helper/main/src/dota2_darkcarnival/assets/img/fool_token.png)
    ![](https://raw.githubusercontent.com/Zirpon/douyu-helper/main/src/dota2_darkcarnival/assets/img/fool_wheel_token.png)
    ![](https://raw.githubusercontent.com/Zirpon/douyu-helper/main/src/dota2_darkcarnival/assets/img/lovers_token.png)

-   注意并不是任意代币组合都会指示出英雄 具体参考英雄 所属 代币组合
